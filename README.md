# Sales Forecasting Dashboard

An intelligent sales forecasting system built on the Superstore Sales dataset — 

covering time series forecasting (SARIMA, Prophet, XGBoost), anomaly detection, 

product demand segmentation, and an interactive dashboard.

Live Dashboard: https://sales-forecasting-dashboard-epwbph8f5ovtz5uq8x9fdo.streamlit.app/

Full Analysis Notebook: [analysis.ipynb](./analysis.ipynb)

Executive Business Report: [summary.docx](./summary.docx)

## What's in this repo

- app.py — Streamlit dashboard (4 pages: Sales Overview, Forecast Explorer, Anomaly Report, Product Segments)

- analysis.ipynb — full data analysis, all forecasting models, anomaly detection, and clustering

- train.csv — Superstore sales dataset

- requirements.txt — Python dependencies

- summary.docx — executive report for business stakeholders

- charts/ — all generated chart images
